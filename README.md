# noa-ats-assignment

노아에이티에스(주) 2026 신입사원 채용 - 사전과제

## 실행 방법

```bash
# 패키지 설치
npm install

# 로컬 개발 서버 실행 (http://localhost:3000)
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm run start
```

## Vercel 배포

1. [Vercel](https://vercel.com)에 가입/로그인
2. GitHub에 이 저장소 푸시 후 Vercel에서 "Import Project"로 연결
3. 또는 Vercel CLI: `npx vercel` (프로젝트 루트에서 실행)
